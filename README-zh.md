## 9. 贡献指南

欢迎贡献！如果你有优秀的 Happy Horse 1.0 提示词，请提交 Pull Request。

### 如何贡献

1. Fork 本仓库
2. 创建新分支 (`git checkout -b feature/add-new-prompt`)
3. 在正确的分类中添加你的提示词（使用下一个可用编号）
4. 提交 PR

**请确保包含以下内容：**
- 完整提示词文本
- 来源链接（原创者的 X/Twitter 帖子）
- 风格/效果简述
- 示例视频（如有）

### 贡献模板

```markdown
### X.X [你的标题]

*简述。*

**提示词：**
```
[你的提示词]
```

**来源：** [创作者名称](https://x.com/@username) - [帖子链接](https://x.com/...)
```
